[role="lead"]
Ab Element 12.0 und den Managementservices Version 2.14 können Sie mithilfe der NetApp Hybrid Cloud Control UI und DER REST-API Firmware-reine Upgrades auf Ihren Storage-Nodes durchführen. Dieses Verfahren führt keine Upgrades für Element Software durch und ermöglicht ein Upgrade der Storage-Firmware außerhalb einer größeren Version.



== Was ist neu

Das Storage-Firmware-Bundle Version 2.76 enthält verschiedene Fehlerbehebungen und neue Funktionen, einschließlich Schwachstellenkorrekturen. Sie können die Speicher-Firmware mithilfe der Anweisungen in aktualisieren link:task_hcc_upgrade_storage_firmware.html["Firmware für Storage-Upgrades"].

== Änderungen in den Komponenten der Speicher-Firmware

|===


| Komponente | Verbesserung/Änderung 


| H610S BIOS 3B06  a| 
* Verbesserte Handhabung von Speicherausfällen
* Microcode-Version auf MCU x6906 aktualisiert




| H610S BMC 3.86.07  a| 
* BIOS-PPR-Ereignisprotokollanalyse in BMC SEL hinzugefügt
* Vermeiden Sie eine Überhitzung des Systems, wenn der BMC aufhängt
* Die Länge der Gruppen-Domain von LDAP wurde von 64 auf 128 Zeichen geändert




| H410S/H610S NIC, MELLANOX 14.25.1020 | K. A. 


| Verschiedene Aktualisierungen Der Laufwerk-Firmware | K. A.

== Gelöste und bekannte Probleme

Siehe https://mysupport.netapp.com/site/bugs-online/product["Bugs Online Tool"^] Für Details zu Problemen, die behoben wurden, und für neue Probleme.



=== Zugriff auf das BOL-Tool

. Navigieren Sie zum  https://mysupport.netapp.com/site/bugs-online/product["BOL-Werkzeug"^] Und wählen Sie in der Dropdown-Liste * Element Software* aus:
+
image::bol_dashboard.png[Versionshinweise für Storage-Firmware-Bundles]

. Geben Sie im Suchfeld „Speicher-Firmware-Paket“ ein, und klicken Sie auf *Neue Suche*:
+
image::storage_firmware_bundle_choice.png[Versionshinweise für Storage-Firmware-Bundles]

. Es wird eine Liste der behobenen oder offenen Fehler angezeigt. Sie können die Ergebnisse wie folgt verfeinern:
+
image::bol_list_bugs_found.png[Versionshinweise für Storage-Firmware-Bundles]
